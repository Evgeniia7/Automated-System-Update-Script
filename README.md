# Automated System Update Script

A Bash script to automate system maintenance tasks. It checks for updates, performs necessary installations, and cleans up old packages. Designed for user-friendly execution on Debian/Ubuntu-based systems.

## Features
- Automated updates using `apt update` and `apt upgrade`
- Cleanup of unnecessary packages and files
- Logging of update success or failure

## Requirements
- Linux (Debian/Ubuntu-based)
- Bash shell
- Sudo privileges

## Getting Started

### 1. Clone the repository
git clone https://github.com/Evgeniia7/Automated-System-Update-Script.git  
cd Automated-System-Update-Script

### 2. Make the Script Executable
chmod +x system_update.sh

### 3. Run the Script
sudo ./system_update.sh
